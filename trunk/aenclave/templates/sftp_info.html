You can use SFTP to upload many songs to Audio Enclave at once.  This is much more efficient if you want to upload an entire album, for instance.  If you just have a single song to upload, it will probably be easier to use the HTTP upload page instead.

How to upload songs using SFTP

  1. Install a working SFTP client.  You can download SecureFX or Fetch from the IS&T software page.
  2. Connect to Nice-Rack with these settings:
    • Host: nice-rack.mit.edu
    • Username: upload
    • Port: 22
    • Protocol: SFTP
    Ask RJ, Femtomatt, Martí, or some other Putz resident for the password.
  3. Upload the songs; it’s okay to have files in subdirectories.  You may only upload MP3 files.  Please do not upload songs that are already in the database.
  4. After you have uploaded everything, click the button below{% if not user.is_authenticated %} (after first logging in){% endif %} and follow the instructions.
  5. Rock out.
